- Descripción del Equipo

Somos el equipo "Juegos Samir", integrado por Erick, Saul y Pablo. Nuestro objetivo es desarrollar juegos educativos y entretenidos para fomentar el aprendizaje mediante la tecnología.

- Proyecto

Nuestro proyecto consiste en crear una plataforma web interactiva con juegos lúdicos enfocados en matemáticas, lógica y programación para estudiantes de primaria y secundaria. Utilizamos HTML, CSS y JavaScript como base tecnológica.
